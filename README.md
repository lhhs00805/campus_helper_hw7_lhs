# campus_helper_hw7_lhs

## 📌 Homework 7 — My Own Python Package

이 리포지토리는 Homework 7에서 요구하는 나만의 Python 패키지 제작 과제를 위해 만들어졌습니다.  
VS Studio Code에서 직접 패키지 구조를 만들고, 로컬에서 실행해보고, 그 결과를 GitHub에 업로드했습니다.

---

## 📦 Package Structure


campus_helper_hw7_lhs/
 ├── __init__.py
 ├── example_main.py
 ├── gcalc.py
 └── schedule.py


- `gcalc.py` — 간단한 성적 계산 기능  
- `schedule.py` — 요일별 공부 루틴 관리 기능  
- `__init__.py` — 외부에서 패키지처럼 import할 수 있게 연결  
- `example_main.py` — 패키지 실행 예시 스크립트
